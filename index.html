Perjalanan Sang Pelajar

Halo! Saya adalah seorang developer web yang sangat menyukai dunia teknologi dan desain antarmuka. Terinspirasi dari keindahan serta detail UI pada game Honkai: Star Rail, saya membangun setiap proyek web dengan orientasi pada performa dan estetika visual.

Saya percaya bahwa kode yang baik adalah kode yang tidak hanya berfungsi secara logika, namun juga memberikan kesan pertama yang memukau bagi pengguna. Terutama animasi dan glassmorphism yang responsif!

  • Lokasi: Indonesia (Bumi)
  • Pendidikan: Teknik Informatika
  • Fokus: Frontend Web Development dan desain grafis
  • Hobi: Coding, Main Game, UI Design
